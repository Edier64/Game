# El Hoyo — Documentación Oficial

Bienvenido a la documentación oficial del videojuego **El Hoyo**, un juego de terror *low poly* en primera persona desarrollado en **Unity**.

Esta documentación está organizada como Manual de Usuario y cubre desde los requisitos mínimos del sistema hasta las mecánicas de juego.
