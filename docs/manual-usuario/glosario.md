# 6. Glosario de Términos

A continuación se definen los términos técnicos y conceptos utilizados en el presente manual, con el objetivo de facilitar la comprensión del documento a todo tipo de usuarios.

| Término | Definición |
|---|---|
| **Unity** | Motor de desarrollo de videojuegos multiplataforma, utilizado para crear el juego *El Hoyo*. |
| **MVC** | Patrón de arquitectura de software que separa la lógica en tres componentes: Modelo (datos), Vista (interfaz) y Controlador (lógica de negocio). |
| **FPS** | *First Person Shooter* o juego en primera persona; género en el que la cámara representa la visión del personaje del jugador. |
| **HUD** | *Heads-Up Display*; conjunto de elementos visuales superpuestos en pantalla que proporcionan información al jugador sin interrumpir la partida. |
| **IA / AI** | Inteligencia Artificial; sistema programado que controla el comportamiento del enemigo (la araña) de forma autónoma y reactiva. |
| **Low Poly** | Estilo visual que utiliza una cantidad reducida de polígonos, generando una estética geométrica y estilizada. |
| **Asset** | Recurso digital utilizado en el juego: modelos 3D, texturas, sonidos, animaciones o scripts. |
| **Feature** | Funcionalidad o módulo específico del sistema; en este proyecto cada mecánica (linterna, araña, jugador) es una feature independiente. |
| **Script** | Archivo de código fuente escrito en C# que define el comportamiento de un elemento del juego dentro de Unity. |
| **Escena** | Espacio virtual de Unity que contiene todos los objetos, luces, cámaras y lógica de una etapa o pantalla del juego. |
| **Game Over** | Estado del juego que se activa cuando el jugador pierde; finaliza la partida actual. |
| **Overlay** | Efecto visual superpuesto sobre la pantalla del juego, como el efecto rojo de tensión al ser detectado por el enemigo. |
| **DirectX** | Conjunto de interfaces de programación de Microsoft utilizadas para renderizar gráficos en Windows. |
| **Bootstrap** | Componente del proyecto que inicializa y configura todos los sistemas del juego al arrancar. |
