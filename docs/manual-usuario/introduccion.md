# Introducción

## Descripción del sistema

**El Hoyo** es un videojuego de terror en primera persona con estética *low poly*, desarrollado en el motor gráfico **Unity**. El jugador debe sobrevivir y escapar de un entorno hostil evitando ser detectado por una araña enemiga con inteligencia artificial.

El juego ha sido desarrollado siguiendo una arquitectura **MVC (Modelo–Vista–Controlador)** adaptada a Unity, lo que permite una estructura de código limpia, escalable y fácil de mantener.

## Objetivo del manual

Este manual de usuario tiene como objetivo proporcionar al usuario final toda la información necesaria para:

- Instalar y ejecutar correctamente el juego.
- Comprender la interfaz y los elementos visuales de la pantalla.
- Conocer los controles y mecánicas de juego.
- Identificar y resolver problemas comunes durante la ejecución.

## Audiencia objetivo

Este manual está dirigido a cualquier persona que desee jugar *El Hoyo*, independientemente de su nivel de experiencia con videojuegos. No se requieren conocimientos técnicos previos.
