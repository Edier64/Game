# 2. Requisitos del Sistema

Antes de proceder con la instalación y ejecución de **El Hoyo**, es necesario verificar que el equipo del usuario cumple con los requerimientos mínimos de hardware y software establecidos para garantizar el correcto funcionamiento de la aplicación.

## 2.1. Requisitos mínimos de hardware

Los requisitos mínimos representan las especificaciones más bajas con las que el juego puede ejecutarse, aunque es posible que el rendimiento no sea óptimo en todos los escenarios.

| Componente | Especificación mínima |
|---|---|
| Sistema operativo | Windows 10 (64 bits) |
| Procesador | Intel Core i3 de 3ª generación o equivalente |
| Memoria RAM | 4 GB |
| Tarjeta gráfica | NVIDIA GeForce GTX 950 / AMD Radeon RX 560 |
| Almacenamiento disponible | 500 MB |
| DirectX | Versión 11 |
| Resolución mínima | 1280 × 720 px |

## 2.2. Requisitos recomendados de hardware

Para una experiencia de juego fluida y con todos los efectos visuales activos, se recomienda el siguiente equipamiento:

| Componente | Especificación recomendada |
|---|---|
| Sistema operativo | Windows 10 / 11 (64 bits) |
| Procesador | Intel Core i5 de 8ª generación o superior |
| Memoria RAM | 8 GB o más |
| Tarjeta gráfica | NVIDIA GeForce GTX 1060 / AMD RX 580 |
| Almacenamiento disponible | 1 GB |
| DirectX | Versión 12 |
| Resolución recomendada | 1920 × 1080 px (Full HD) |

## 2.3. Requisitos de software

| Software | Requerimiento |
|---|---|
| Sistema operativo | Windows 10 o superior (64 bits) |
| .NET Framework | 4.x o superior (incluido en Windows 10/11) |
| Visual C++ Redistributable | 2019 o superior |
| Unity Editor | Solo necesario para desarrollo; no requerido para jugar |

> **Nota importante:** El juego no ha sido probado oficialmente en sistemas operativos macOS ni Linux. Se recomienda su uso exclusivo en entornos Windows. El uso en otros sistemas operativos queda bajo responsabilidad del usuario.

## 2.4. Periféricos necesarios

- **Teclado:** Estándar (se utilizan teclas WASD, F, E, C, Shift y Escape).
- **Mouse:** Necesario para controlar la cámara y navegar los menús.
- **Monitor:** Con resolución mínima de 1280 × 720 píxeles.
- **Altavoces o auriculares:** Recomendados para una experiencia de terror completa.
