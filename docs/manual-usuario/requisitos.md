# Requisitos del Sistema

Antes de instalar y ejecutar **El Hoyo**, verifica que tu equipo cumpla con los siguientes requisitos mínimos y recomendados.

## Requisitos mínimos

| Componente | Especificación mínima |
|---|---|
| Sistema operativo | Windows 10 (64 bits) |
| Procesador | Intel Core i3 o equivalente |
| Memoria RAM | 4 GB |
| Tarjeta gráfica | NVIDIA GTX 950 / AMD RX 560 |
| Almacenamiento | 500 MB disponibles |
| DirectX | Versión 11 |

## Requisitos recomendados

| Componente | Especificación recomendada |
|---|---|
| Sistema operativo | Windows 10/11 (64 bits) |
| Procesador | Intel Core i5 o superior |
| Memoria RAM | 8 GB |
| Tarjeta gráfica | NVIDIA GTX 1060 / AMD RX 580 |
| Almacenamiento | 1 GB disponibles |
| DirectX | Versión 12 |

## Software necesario

- **Unity Runtime** (incluido en el ejecutable compilado)
- No se requiere instalación de Unity Editor para jugar la versión de distribución.

> **Nota:** El juego no ha sido probado oficialmente en macOS o Linux. Se recomienda el uso exclusivo en Windows.
